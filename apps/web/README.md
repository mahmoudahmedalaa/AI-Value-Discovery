# AI Value Discovery Web App

Browser-based internal partner demo for AI Value Discovery.

## Commands

```bash
npm run dev
npm run lint
npm run typecheck
npm run build
```

The app uses seeded GCC bank demo data and deterministic mock AI posture. It does not require paid services for the first demo.

## Current Routes

- `/` workspace selector
- `/executive-cockpit`
- `/opportunities`
- `/opportunities/[id]`
- `/intake`
- `/documents`
- `/portfolio`
- `/decision-packs`
- `/methodology`
- `/audit`
- `/security-posture`

The app should stay local-first until the internal demo flow is complete.

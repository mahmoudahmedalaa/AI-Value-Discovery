# User Journeys

## Internal partner demo

Open local app, select sample GCC bank workspace, show executive cockpit, run mock extraction, review opportunities, inspect detail, show evidence/assumptions/missing info, score, portfolio, decision pack, audit, security posture.

## Consultant discovery sprint

Create workspace, configure sector/policy, upload materials, extract opportunities, review drafts, enrich fields, score, validate with client, finalize decision pack.

## CIO review

Filter high-value opportunities, review data/system dependencies, add readiness notes, approve or flag.

## Risk review

Review high-risk opportunities, add controls, block/approve, audit event recorded.

## ExCo decision

Review cockpit, top recommendations, approve Fund Now list, assign Fix First actions, export pack.

# UX Copy Guide

## Voice

Serious, clear, useful, direct.

## Use

Opportunity, portfolio, funding readiness, evidence strength, control exposure, data blocker, baseline missing, human review required.

## Avoid

Magic, supercharge, unlock, revolutionary, game-changing, autonomous consultant, replace your strategy team.

## Tooltip examples

Evidence strength: measures how strongly this opportunity is supported by source documents, user inputs, or verified data.

Data readiness: measures whether required data exists, has an owner, can be accessed, and is usable for pilot.

Control exposure: measures privacy, regulatory, cyber, model-risk, explainability, audit, and human-review requirements.

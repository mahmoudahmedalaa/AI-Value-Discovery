# Design System

## Design personality

Enterprise, calm, sharp, credible, executive-grade.

## Avoid

Neon gradients, purple AI glow, glassmorphism, excessive chips, emoji-heavy UI, generic AI visuals, decorative graphs, chatbot-first design.

## Prefer

Restrained neutral palette, strong typography, clean grid, dense readable tables, clear hierarchy, elegant status badges, conservative charts, subtle borders, useful tooltips.

## Semantic states

Fund Now, Fix First, Defer, Stop, Explore Further, Unreviewed.

## Components

Cards, tables, score indicators, decision badges, tooltips, forms, modals, evidence labels, assumption labels, AI-generated labels, empty/loading/error states.

## Dashboard rule

Every chart must support a decision. No decoration-only charts.

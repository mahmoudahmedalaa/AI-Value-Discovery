# Wireframe Spec

## Global layout

Left sidebar, top bar with workspace selector and deployment badge, main content area, optional right panel for evidence/methodology.

## Executive Cockpit

Header, KPI cards, portfolio decision chart, top opportunities table, readiness blockers, required decisions.

## Intake Hub

Upload zone, manual opportunity button, import Excel button, processing jobs, draft extraction queue, classification reminder.

## Opportunity Backlog

Filterable table: name, function, workflow, value lever, owner, fundability, decision, evidence, data readiness, control exposure.

## Opportunity Detail

Summary, problem, workflow/users, value/KPIs, data/systems, risk/controls, scores, evidence, assumptions, missing info, recommendation, audit.

## Portfolio Cockpit

Decision lanes, matrix, risk heatmap, roadmap, table.

## Security Posture

Deployment mode, data boundary diagram, storage, AI inference, logging, classification, retention.

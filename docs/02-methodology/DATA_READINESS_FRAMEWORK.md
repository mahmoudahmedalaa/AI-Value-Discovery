# Data Readiness Framework

Assess data existence, ownership, accessibility, quality, lineage, sensitivity, refresh frequency, integration effort, consent/legal basis, and retention/deletion.

## Score

1 = data path unknown. 2 = data exists but access/quality unclear. 3 = available with manageable issues. 4 = usable with minor remediation. 5 = ready for pilot.

## Required output

Each opportunity should identify required data assets, source systems, owners, sensitivity, quality issues, access blockers, score, and remediation actions.

# Scoring Model

## Dimensions, scored 1–5

- Strategic alignment
- Value potential
- Evidence strength
- Data readiness
- Workflow readiness
- Technical feasibility
- Control score
- Adoption complexity
- Sponsor strength
- Time-to-impact
- Reusability

## Suggested weights

Strategic alignment 10%, value 18%, evidence 12%, data readiness 12%, workflow 8%, technical feasibility 8%, control score 10%, adoption 7%, sponsor 7%, time-to-impact 5%, reusability 3%.

## Decision logic

### Fund Now
Composite >= 75, value >= 4, evidence >= 3, data readiness >= 3, sponsor >= 4, KPI and owner exist, controls manageable.

### Fix First
High value but blockers exist in data, controls, owner, workflow, or evidence.

### Defer
Moderate score, value unclear, timing not right.

### Stop
Low score, no KPI/owner, high risk/low value, generic chatbot anti-pattern, or duplicative.

### Explore Further
Potentially interesting but insufficient evidence.

## Explanation requirement

Every recommendation must explain rationale, evidence, assumptions, blockers, next actions, and confidence.

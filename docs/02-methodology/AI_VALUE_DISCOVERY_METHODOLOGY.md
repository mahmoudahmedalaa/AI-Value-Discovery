# AI Value Discovery Methodology

## Stages

1. Demand capture from executive priorities, department ideas, pilots, vendor proposals, strategy docs, workshop notes, process pain points, KPI reports, and data catalogues.
2. Normalization into standard opportunity schema.
3. Evidence classification: source-backed evidence, user assumption, AI inference, consultant judgment, missing info.
4. Value assessment using ranges and confidence.
5. Readiness assessment across data, workflow, systems, owner, and controls.
6. Control assessment across privacy, cyber, regulatory, model risk, audit, human review, and data residency.
7. Portfolio decisioning: Fund Now, Fix First, Defer, Stop, Explore Further.
8. Output generation: heatmap, summary, pilot charters, blocker register, roadmap.

## Human review gates

Required before score override, decision approval, final decision-pack generation, and any Fund Now recommendation.

## Methodology rule

The platform should challenge weak ideas instead of making every idea look good.

# Value Levers and KPI Library

## Revenue uplift

KPIs: conversion rate, revenue per customer/RM, cross-sell rate, churn, average deal size.

## Cost reduction

KPIs: cost per case, processing time, FTE hours, rework, automation rate.

## Productivity

KPIs: cycle time, cases per employee, hours saved, turnaround time.

## Risk reduction

KPIs: false positives, false negatives, loss avoided, control breach count, time to detection.

## Customer experience

KPIs: NPS, CSAT, complaints, first contact resolution, average handling time.

## Compliance improvement

KPIs: exceptions, audit findings, review time, evidence completeness.

## Rules

Do not invent baseline values. Use ranges. Separate hard financial value from soft/strategic value. Show assumptions and missing inputs.

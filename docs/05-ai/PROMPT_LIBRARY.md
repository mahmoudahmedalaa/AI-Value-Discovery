# Prompt Library

Prompts must be versioned and tested.

## Extract opportunities

Only extract source-supported opportunities. Do not invent KPI values. Separate evidence from assumptions. Output JSON only.

## Normalize opportunity

Identify workflow, user, problem, KPI, value lever, data, controls, and missing fields.

## Identify controls

Consider privacy, cyber, regulatory, model risk, explainability, audit, human-in-loop, data residency.

## Challenge opportunity

Be direct. Identify weak assumptions, missing evidence, and what must be fixed before funding.

## Generate decision pack

Use approved opportunities only. Separate evidence and assumptions. Avoid hype.

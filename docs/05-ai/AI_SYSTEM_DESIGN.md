# AI System Design

AI assists with extraction, synthesis, scoring support, challenge, and report drafting. It does not silently make final investment decisions.

## Services

Document extraction, opportunity extraction, opportunity normalization, KPI/value agent, data readiness agent, risk/control agent, challenger agent, portfolio recommendation agent, decision-pack generation agent.

## Rules

Use structured output schemas, validate all AI output, store evidence links, mark AI-generated fields, require human review for client-facing outputs.

## Pre-MVP

Use deterministic mock responses from fixtures.

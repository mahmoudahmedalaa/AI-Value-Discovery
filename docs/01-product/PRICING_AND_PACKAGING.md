# Pricing and Packaging

Start with productized consulting powered by proprietary software, not low-cost SaaS.

## Packages

1. Internal Demo: not sold, synthetic data, local demo.
2. AI Value Discovery Sprint: AED 75k–150k, 10 working days, portfolio and decision pack.
3. Enterprise Workspace: ongoing platform access after sprint.
4. Client-hosted Deployment: setup fee + annual license/support.
5. Implementation Bridge: pilot scoping, architecture, data readiness, governance, delivery support.
6. Partner/White-label Edition: branded version for boutiques/SIs.

## Expansion path

Discovery sprint -> pilot charter -> data readiness -> governance -> implementation -> ongoing portfolio management.

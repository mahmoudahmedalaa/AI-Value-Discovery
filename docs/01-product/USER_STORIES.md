# User Stories

## Workspace

As a Mal7 consultant, I want to create a client workspace so all opportunities, documents, scores, and outputs are organized under one engagement.

Acceptance: workspace has name, sector, client type, deployment mode, objectives, unique ID, and isolated data.

## Opportunity

As a strategy director, I want to create an AI opportunity manually so workshop ideas can be captured immediately.

Acceptance: form captures problem, workflow, user, value lever, KPI, owner, data assets, systems, controls, assumptions, and notes.

## Scoring

As a consultant, I want the platform to score an opportunity so I can compare it against others.

Acceptance: dimension scores, composite score, recommendation, and explanation are visible.

## Portfolio

As an executive sponsor, I want to see opportunities grouped by decision state so I can make funding decisions.

Acceptance: Fund Now/Fix First/Defer/Stop/Explore Further views exist with filters.

## Document intelligence

As a consultant, I want AI to extract candidate opportunities from documents so manual synthesis is faster.

Acceptance: AI creates draft opportunities with evidence references and requires human approval.

## Decision pack

As a strategy director, I want to generate an executive pack so leadership can review the AI portfolio.

Acceptance: pack includes summary, heatmap, top opportunities, blockers, risks, and roadmap.

# MVP Scope

## Phase 1 — Internal Demo / Pre-MVP

Included: local app, sample GCC bank workspace, enterprise UI shell, seeded documents/opportunities, manual opportunity management, mock AI extraction, scoring engine, portfolio cockpit, decision-pack preview, audit log, security posture, demo script.

Excluded: real LLM API dependency, production auth, production deployment, enterprise integrations, real client data.

## Phase 2 — MVP Product Spine

Included: multi-workspace support, real document upload, Postgres schema, Qdrant indexing, MinIO object storage, human review queue, exports, basic RBAC, audit log.

## Phase 3 — AI/RAG MVP

Provider abstraction, document extraction, embeddings, structured AI output validation, evidence retrieval, AI eval fixtures.

## Phase 4 — Pilot-Ready Enterprise MVP

SSO-ready architecture, tenant isolation tests, retention controls, deployment docs, procurement FAQ, client-hosted deployment path.

# Roadmap

## Phase 0 — Product Governance & Repo Foundation
Docs, GitHub board, labels, milestones, repo structure, ADRs, setup plan.

## Phase 1 — Internal Demo / Pre-MVP
Local app, design shell, GCC bank seed data, opportunity CRUD, mock AI, scoring, portfolio, decision-pack preview, audit/security posture, demo script.

## Phase 2 — MVP Product Spine
Persistent DB, schemas, document upload, exports, RBAC, audit.

## Phase 3 — AI/RAG & Document Intelligence
Qdrant, chunking/embeddings, provider abstraction, structured extraction, evidence linking, AI evals.

## Phase 4 — Portfolio Cockpit & Decision Packs
Advanced charts, roadmap, financial panels, PPTX/PDF/Excel exports, pilot charters.

## Phase 5 — Enterprise Security & Deployment
Tenant isolation tests, client-hosted docs, SSO-ready auth, retention, procurement FAQ, threat model implementation.

## Phase 6 — Pilot-Ready Release
Hardened MVP, demo environment, onboarding checklist, security pack, client-ready templates.

# Procurement Security FAQ

Where is data stored? Depends on deployment mode; regulated clients can deploy inside client cloud/VPC.

Is data used for training? No by product policy; provider terms must be validated.

Can client use own AI provider? Yes, via model abstraction.

Can it run without external LLM calls? Yes, restricted deployments should support local/client-managed models.

Are embeddings client data? Yes.

Can data be deleted? Yes, including derived artifacts.

Is there audit? Yes.

Can it be client-hosted? Yes.

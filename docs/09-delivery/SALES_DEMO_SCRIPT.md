# Sales Demo Script

Opening: “Most organizations do not need more AI ideas. They need a disciplined way to decide which ideas deserve funding.”

Flow: sample GCC bank workspace, executive cockpit, intake/mock extraction, extracted opportunities, opportunity detail, evidence/assumptions, scoring, portfolio, decision pack, security posture.

Closing: “This is not a chatbot. It is the decision system that tells leadership where AI money should go and what must be fixed before scaling.”

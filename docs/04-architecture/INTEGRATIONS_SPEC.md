# Integrations Spec

## Pre-MVP

No external integrations required.

## Later

SharePoint, OneDrive, Google Drive, Confluence, Jira, Azure DevOps, ServiceNow, Power BI, Tableau, Snowflake, Databricks, Salesforce, Dynamics, SAP, Oracle, Entra ID, Okta, SAML/OIDC.

## Principle

Integrations must respect deployment boundary and retain source metadata.

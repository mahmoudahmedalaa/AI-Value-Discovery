# Test Strategy

Unit tests: scoring, decision logic, data policy, schema validation.

Integration tests: API routes, DB operations, document lifecycle, AI provider gateway, portfolio aggregation.

E2E tests: workspace selection, create opportunity, score, portfolio, decision pack, audit.

AI eval tests: fixtures, structured output, hallucination checks.

Security tests: workspace isolation, permissions, restricted documents, unauthorized API.

Pre-MVP minimum: scoring unit tests, demo E2E, workspace isolation tests.

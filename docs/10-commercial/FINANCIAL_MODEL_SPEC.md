# Financial Model Spec

Inputs: baseline KPI, target KPI, volume, unit cost, revenue per unit, FTE cost, cycle time, expected improvement, implementation cost, run cost, horizon, confidence.

Outputs: conservative/base/upside value, payback, implementation cost, net value, ROI, sensitivity drivers, confidence.

Portfolio: total value range, value by function/lever, investment required, risk-adjusted value, time-to-impact, fund-now value, fix-first trapped value.

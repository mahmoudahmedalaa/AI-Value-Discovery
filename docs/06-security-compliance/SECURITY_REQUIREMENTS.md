# Security Requirements

Authentication, RBAC, tenant/workspace isolation, encryption in transit/at rest, file validation, audit logging, input validation, rate limiting, secrets management, secure headers, safe errors, deletion workflows.

Roles: Mal7 Super Admin, Client Admin, Consultant, Business Reviewer, Risk Reviewer, Executive Viewer, Read-only Viewer.

Pre-MVP must model roles, workspace boundaries, audit events, document classification, and data policy.

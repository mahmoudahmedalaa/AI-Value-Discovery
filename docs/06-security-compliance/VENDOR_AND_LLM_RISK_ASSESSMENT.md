# Vendor and LLM Risk Assessment

Assess data residency, data use/training, subprocessors, encryption, retention, logging, access controls, certifications, availability, contractual terms, and client acceptance.

Provider options: mock, local, OpenAI, Azure OpenAI, Bedrock, Vertex, client-managed.

The platform must answer: where data is stored, where inference happens, whether prompts are stored, whether data trains models, whether client can use own cloud/model, deletion and audit capabilities.
